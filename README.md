# Yle Areena Embed
Yle Areena Embed

## Usage

```html
<iframe frameborder="0" allowfullscreen
  src="https://theel0ja.github.io/yle-areena-embed/index.html?id=VIDEO_ID"
  width="WIDTH" height="HEIGHT"></iframe>
```

Replace `VIDEO_ID`, `WIDTH` and `HEIGHT` with your own values.
